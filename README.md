# NodeJS MVC app

## Setup

1. Install dependencies
```bash
npm install
```

2. Rename `.env.example` to `.env`

3. Start the server
```bash
npm run start
```

If you wish to start the server in **development mode** using `nodemon`
```bash
npm run start:dev
```
